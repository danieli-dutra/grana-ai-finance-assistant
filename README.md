# 💰 Grana.ai (Archive)

> ⚠️ Este repositório foi arquivado e não recebe mais atualizações.

O Grana.ai começou como um experimento de **Vibe Coding** utilizando a plataforma Lovable durante um laboratório da DIO.

Ao longo do desenvolvimento, o projeto evoluiu significativamente, deixando de ser apenas um protótipo para se tornar uma aplicação em desenvolvimento ativo.

## 🚀 O projeto continua aqui

O desenvolvimento oficial foi migrado para um novo repositório:

👉 **https://github.com/danieli-dutra/grana-ai**

Esse novo repositório representa a próxima fase do projeto, incluindo:

- desenvolvimento local com React + TypeScript;
- evolução da arquitetura da aplicação;
- controle completo do código-fonte;
- colaboração em equipe;
- documentação técnica contínua;
- preparação para o Projeto Final da trilha Full Stack (+praTi & Codifica).

## 📌 Sobre este repositório

Este repositório permanece público por preservar uma parte importante da história do Grana.ai.

Ele documenta:

- as primeiras versões do produto;
- a fase de experimentação com IA e Vibe Coding;
- a evolução do conceito;
- decisões iniciais de UX, produto e arquitetura.
